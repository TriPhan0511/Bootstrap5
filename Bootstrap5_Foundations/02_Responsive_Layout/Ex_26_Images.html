Images

Responsive images

Images in Bootstrap are made responsive with .img-fluid. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent element.

Image thumbnails

In addition to our border-radius, you can use .img-thumbnail to give an image a rounded 1px border appearance.

Aligning images

Aligning images with the helper float classes (.float-start, .float-end, float-none...) or text alignment classes (.text-start, .text-end, .text-center, ...). block-level images (using .d-block utility class) can be centered using the .mx-auto margin utility class.

Circle image

If you want to make a circle image, you have to use an equal image (width = height) and .rounded-circle class.
