Bootstrap Margin

Margins are the space around the border of a block element. The bigger the margin, the more space is assigned.

Margins are called using the m-1, m-2, m-3, m-4 and m-5 classes. And m-0 is the reset.

You can control the direction of margin, assigning it to top, bottom, start and end as you do with border.

The margins below are set using the classes:

  • m-* : sets the complete margin
  • ms-* : sets the start margin (left margin)
  • me-* : sets the end margin (right margin)
  • mt-* : sets the top margin
  • mb-* : sets the bottom margin
  • mx-* : sets both start and end margins (x-axis)
  • my-* : sets both top and bottom margins (y-axis)

They take sizing 1 to 5 as the previous set did.
