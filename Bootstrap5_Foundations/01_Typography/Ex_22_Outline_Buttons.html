Outline Buttons

In addition to the normal buttons, you can create buttons with an outline style instead of a solid fill. There's no particular advantage to create outline buttons beyond styling.

The outline buttons handle color changes when the mouse moves over them

The outline button is initially set up with the .btn class, then a color is assigned using the standard color names.

The color of the text inside the button is the same as the outline of the button.

Outline Button Group

The buttons are placed in a single div that is given the .btn-group class, then the buttons are created as normal. Rather than being separate from one another, they appear side by side.
