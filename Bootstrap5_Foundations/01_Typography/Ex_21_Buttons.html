Buttons

Buttons are a very common UI element of Bootstrap, and their style can be used on hyperlinks, forms, diaglog boxes and more.

Buttons can be created from multiple elements, including <a>, <input> and <button>. Even <div> could be used, thought it should only be used where the user is required to interact with that part of the website.

The button is initially set up with a .btn class, then a color is assigned using the standard color names.

  • .btn-primary
  • .btn-secondary
  • .btn-success
  • .btn-info
  • .btn-warning
  • .btn-danger
  • .btn-dark
  • .btn-light

The color of the text inside the button is selected using the .text-* classes, which accept the same standard Bootstrap color names.

In addtition to those, we have the .btn-link class, which make the buttons look like a standard link - however, it is rarely used. The <a> element performs this action with the same styling, though it is not a block element.

Button Groups

We can group buttons together so that they form a united panel. This is useful when we want controls or buttons presented as a choice rather than individual options.

The buttons are placed in a single div that is given the .btn-group class, then the buttons are created as normal. Rather than being separated from one another, they appera side by side.
